서비스

아이디어를 빠르게 형태로 만들고, 데이터를 통해 다음 스텝을 결정합니다.

MOBILE • ANDROID / iOS

모바일 앱 설계 & 개발

네이티브/크로스 플랫폼 앱을 목표 지표에 맞춰 개발합니다. KTX 아키텍처, Clean Architecture, Compose/SwiftUI 적극 활용.

WEB • SAAS

경량 웹 제품

정적 사이트부터 대시보드까지. SEO · 접근성 · 속도를 기본값으로, 필요한 것만 담습니다.

OPS • DEVEX

배포/운영 자동화

GitHub Actions로 CI/CD, Sentry/CloudWatch로 모니터링, IaC로 재현 가능한 환경을 구성합니다.

레퍼런스

BeanNote

메모 앱 프로토타입 → 3주 내 베타 런칭. 사용자 세션 분석으로 온보딩 이탈 32% 감소.

Bubble Admin

경량 관리자 대시보드. 정적 호스팅 + API 연동으로 비용 80% 절감.

Realtime Chat POC

WebSocket/FCM 하이브리드 채팅 POC. 모바일에서 배터리 사용량 18% 절감.
